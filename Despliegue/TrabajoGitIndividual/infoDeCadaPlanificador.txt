TRELLO Version Gratuita:(me gusta que es facil usar el premium sin pagar)
    * Colaboradores limitados 
    * No tenemos dispoibles todas las funcionalidades de vista (calendario, cronograma, panel, tabla)
    * Puedes crear cuentas muy facilmente y activar la version de prueba de 2 semanas sin usar tarjeta
    * Utilizando temp-mail puedes estar infinitamente usando la version premium dado que solo piden una verificacion
        muchas plantillas comodas, calendario y cronologia(me encanta como se ve) intuitivo para los empleados con secciones de comentarios, y un panel que indica el progreso
    * facil monitorear actividad de cada usuario con tan solo pulsar "ver actividad"
    * informes de correo electrónico de forma automatica 
    * crear reglas que realicen acciones de forma automatica, crear fechas de vencimiento de una tarjeta de trello etc
    * de primera instancia la interfaz es un poco confusa
    * para prevenir perder el tablero le damos perimisos a la otra cuenta de tomar todo el espacio de trabajo (esto teoricamente subira la factura), factura que nunca llegara, y luego ademas esa cuenta hace una copia de los tableros en su propia cuenta
    * problema de este hackeo, tener que reinvitar a todas las cuentas cada 14 dias 
    * super facil de usar

    * capacidad de crear sprints de trabajo siguiendo asi el ciclo de SCRUM muy popularizado 
    * interfaz menos estetica pero funciona igual

    * más orientado a proyectos a gran escala que trello
