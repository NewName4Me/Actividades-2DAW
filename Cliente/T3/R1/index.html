Lista de ejercicios Erik TR 2ºDAW

Ejercicio 1: Función Máximo

Define una función máximo que dados 4 valores devuelva el máximo de ellos. Llámala con un ejemplo pidiendo los 4 valores al usuario.

Ejercicio 2: Lanzamiento de Dado

Crea una web desde la que se simule el lanzamiento de un dado de 6 caras. Para ello define una función "lanzamiento" que devuelva un número aleatorio entre 1 y 6.

Ejercicio 3: Simulación de 6000 Tiradas

Mejora la web anterior para generar una simulación de 6000 tiradas, mostrando al final el número de ocurrencias de cada valor.

Ejercicio 4: Volumen de una Esfera

Crea una web para calcular el volumen de una esfera. Diseña una función que dado el radio de ésta devuelva el volumen.

Ejercicio 5: Área de un Círculo

Mejora el ejercicio anterior permitiendo calcular también el área de un círculo.

Ejercicio 6: Potencias Recursivas

Crea una función para calcular potencias de un modo recursivo.

Ejercicio 7: Factorial de Números

Crea una función que calcule el factorial de un número dado. Diseña una web que muestre en forma de tabla el factorial para los valores de 1 a 10.
