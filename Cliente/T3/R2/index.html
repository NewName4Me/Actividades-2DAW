Lista de Ejercicios Arrays

Ejercicio 1: Array de Números

Realiza un script que solicite 5 números al usuario, almacenarlos en un array y luego mostrar el valor mínimo, máximo y medio.

Ejercicio 2: Comprobar Color en Array

Determina si un color introducido por el usuario está dentro del array ["azul", "amarillo", "rojo", "verde", "café", "rosa"].

Ejercicio 3: Generar Números Aleatorios

Genera 20 números aleatorios y guárdalos en un array. N es introducido por el usuario.

Ejercicio 4: Operaciones con Arrays

Genera los arrays de Intersección, Unión y Diferencia entre dos arrays de 5 palabras cada uno.

Ejercicio 5: Array Aleatorio y Estadísticas

Genera un array con 20 números aleatorios entre 0 y 100, muestra los valores, el mínimo, máximo y promedio.

Ejercicio 6: Buscar Texto en Documento

Crea una web con varios párrafos. Un botón solicitará una palabra y mostrará si existe en el documento.

Ejercicio 7: Ordenar Array con Algoritmo

Genera un array con 20 números aleatorios, muestra los valores sin ordenar, ordenados y sin duplicados utilizando un algoritmo de ordenación como el de burbuja.

Ejercicio 8: Ordenar Nombres

Solicita nombres separados por comas, luego muestra los nombres ordenados utilizando split.

Ejercicio 9: Análisis de Cadena de Texto

Leer una cadena y mostrar: número de palabras, primera y última palabra, palabras en orden inverso, de la A a la Z y de la Z a la A.
